# carlosmartinezt.com

A public journal where I preserve and clarify what I'm learning about growth, relationships, and values. Primarily for my kids, and secondarily for anyone I care about who might need it.

**Live at [carlosmartinezt.com](https://carlosmartinezt.com)**

## What you'll find here

Writing on topics like:

- **AI** — how I use AI as a thinking partner, not a shortcut
- **Career** — frameworks for growth, impact, and navigating work
- **Thinking** — problem articulation, reframing, and clarity
- **Parenting** — lessons from raising two young chess players

## Built with

- [Astro](https://astro.build) — static site generator
- Markdown & MDX for content
- Deployed to GitHub Pages

## Running locally

```bash
npm install
npm run dev       # localhost:4321
npm run build     # production build to ./dist/
```
