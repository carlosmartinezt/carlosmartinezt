# Images Needed

Please add the following images to this directory:

1. **hero.jpg** - NYC skyline or similar background image for the hero section
   - Recommended size: 1920x1080px or larger
   - Should be a high-quality photo

2. **profile.jpg** - Your profile photo
   - Recommended size: 300x300px or larger (will be displayed as 150x150px)
   - Should be a square or will be cropped to circular shape

You can add these images by placing them in this directory.
